# 4️⃣ “Saludo personalizado según hora”

**Objetivo:** condicional múltiple (if / else if / else).

**Enunciado:**
Pide una hora (0 a 23) y muestra:
- 5–11 → “Buenos días”
- 12–18 → “Buenas tardes”
- 19–23 o 0–4 → “Buenas noches”

**Pistas:**
- Convertir la hora a número.
- Varios else if (hora >= 5 && hora <= 11) {} etc.
