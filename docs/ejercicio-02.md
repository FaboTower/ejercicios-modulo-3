# 2️⃣ “¿Aprobó o reprobó?”

**Objetivo:** condicional simple + operador de comparación.

**Enunciado:**
Pide al usuario una nota (1 a 7).
Si la nota es mayor o igual a 4, mostrar: “Aprobado”.
Si es menor a 4, mostrar: “Reprobado”.

**Pistas:**
- Variable nota.
- Condición: if (nota >= 4) { ... } else { ... }.
