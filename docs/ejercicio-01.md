# 1️⃣ “Clasificador de edad simple”

**Objetivo:** variables, tipos number, if/else.

**Enunciado:**
Pide al usuario su edad y muestra:
- “Eres menor de edad”
- “Eres mayor de edad” si es 18 o más.

**Pistas técnicas:**
- Usar prompt + Number() o parseInt().
- Comparar con >= y <.
- Mostrar resultado con console.log o alert.
