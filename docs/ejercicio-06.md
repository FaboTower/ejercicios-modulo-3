# 6️⃣ “Clasificador de notas” (más completo)

**Objetivo:** escalera de condiciones + rangos.

**Enunciado:**
Pide una nota (1 a 7) y muestra:
- [6.0 – 7.0] → “Excelente”
- [5.0 – 5.9] → “Bueno”
- [4.0 – 4.9] → “Suficiente”
- Menos de 4.0 → “Insuficiente”

**Pistas:**
- Usar if (nota >= 6)… else if (nota >= 5)…
- No necesitas poner nota < 6 porque ya lo descartaste arriba.
