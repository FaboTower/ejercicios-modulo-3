# 5️⃣ “Verificador de tipo de dato”

**Objetivo:** usar typeof, trabajar con distintos tipos.

**Enunciado:**
Pide al usuario que ingrese algo (un texto).
Muestra por consola:
- El valor ingresado.
- Qué tipo de dato ve JavaScript con typeof.

Luego, prueba manualmente:
- números (123)
- texto (hola)
- booleanos (true / false sin comillas → usando confirm como variante).

**Pistas:**
- const valor = prompt("...");
- console.log(typeof valor); (siempre string en prompt → sirve para explicar).
